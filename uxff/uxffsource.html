This page provides Software Source and Object Code support links for the UX FFront Unit Project

The UX FFront Unit is an alternate design-spin on the original UX Front Unit used by the ICOM IC-900 UX band modules. It acts as a form-fit-function replacement for the orignal PCB Assembly (PCBA), in the case where the original board is lost or damaged beyond repair. However, with alternate software and assembly components, the module can perform other interesting tasks. A small microcontroller is used to intercept control data and manage the communications of the base controller.
